Como funciona a reprodução automática

  • Os vídeos são configurados com muted e loop para permitir autoplay
  • O JavaScript detecta quando os vídeos estão visíveis na tela
  • Quando um vídeo fica visível, ele começa a reproduzir automaticamente
  • Quando não está mais visível, a reprodução é pausada
  • Clique em qualquer vídeo para dar foco e reproduzir com áudio

Código para reprodução automática

// Configuração básica do vídeo para autoplay
<video loop muted autoplay>
    <source src="video.mp4" type="video/mp4">
</video>

// JavaScript para detectar visibilidade
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        const video = entry.target.querySelector('video');
        if (entry.isIntersecting) {
            video.play();
        } else {
            video.pause();
        }
    });
}, { threshold: 0.5 });

// Observar todos os elementos de mídia
document.querySelectorAll('.media').forEach(media => {
    observer.observe(media);
});
